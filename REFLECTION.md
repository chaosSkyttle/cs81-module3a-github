Reflection: 
What did you learn about using Git and GitHub?
I learned that Git and GitHub can be used for developers to collaborate, by allowing different branches to be developed and merged with the main branch. I also learned that GitHub profiles are a great way to showcase your work and experience. 

What was most confusing or frustrating?
What was most confusing and frustrating for me was using the terminal to commit, because I initially struggled to find files, and received a lot of errors. 

Why do you think developers use Git?
I think developers use Git because it allows for you to gradually commit, and shows you different points in the development process that can be returned to. Additionally, it is helpful for collaboration because of the branching system, which creates  opportunities to test new features in a controlled space. 

How might this help with larger projects in the future?
With larger projects in the future, Git will be helpful because it allows me to see past changes, and return to previous versions. This can be extremely helpful in testing new features of a project.
