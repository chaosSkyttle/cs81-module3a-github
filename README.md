# cs81-module3a-github
Github repo for CS81 Javascript course. 

greet(): used to greet someone by name

isEven(): used to determine whether a number inputted is even

multiplyNum(): returns the product of two numbers

tellTime(): given the hour, minute, and whether it is am/pm, the tellTime() function will return a statement of what time it is.

quantity(): given a number and an object the quantity function will return a statement of how much of the object in possesion.

date(): given the day, month, and year, the date() function will return the date in the month/day/year notation style.
